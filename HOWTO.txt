HOWTO:

1. Clone the git repository
2. copy the css/sass/* files to your project
3. Include follwing files in your application:
     reset.sass
     text.sass
     grid.sass

http://www.divitodesign.com/2008/12/960-css-framework-learn-basics/
Please note that these websites refer to pixels since the orignal 960gs is pixel based.
Fluid 960 Grid System is NOT pixel based
